# Poetry Laravel API

## Установка

Для запуска приложения требуется **Docker** и **Docker Compose**.

Для инициализации приложения выполнить команду:
```
make init
```

## Управление

Запуск:
```
make up
```

Остановка приложения:

```
make down
```

## Интерфейсы

Приложение - http://localhost:8080/api

## Тесты

```
make backend-test
```
